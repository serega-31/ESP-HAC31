1. Назначение и область применения

Настоящее программное обеспечение (прошивка) является объектом академического исследования и разработано в рамках изучения архитектуры микроконтроллеров и анализа защищённости беспроводных протоколов передачи данных.

Целью проекта является изучение методов построения систем защиты информации (СЗИ) и проведение санкционированного анализа защищённости (пентестинга) в изолированных лабораторных условиях. Инструмент предназначен исключительно для специалистов по информационной безопасности, системных администраторов и студентов профильных специальностей.

2.

Функциональные возможности и соответствие исследовательским задачам

Программно-аппаратный комплекс реализует исключительно аналитические функции и функции моделирования штатных сетевых процессов для проверки отказоустойчивости оборудования:

* **Анализ пакетов и аудит BLE (Bluetooth Low Energy):** Пассивный мониторинг радиоэфира в диапазоне 2.4 ГГц, сбор рекламных данных (Advertisement packets) периферийных IoT-устройств. Используется для демонстрации рисков утечки идентификаторов в умных домах.
* **Пассивный мониторинг Wi-Fi (режим Promiscuous 802.11):** Фиксация заголовков управляющих кадров (Management Frames) для оценки загруженности каналов, плотности размещения точек доступа и обнаружения неавторизованных (Rogue) устройств в контролируемом периметре.
* **Моделирование отказоустойчивости и стресс-тестирование:** Программная симуляция внештатных ситуаций на канальном уровне посредством отправки служебных кадров деаутентификации (Deauthentication Frames).

Данный модуль спроектирован для:

* Проверки устойчивости сетевой инфраструктуры к отказам в обслуживании (DoS).
* Анализа применимости механизмов защиты управляющих кадров (стандарт 802.11w / PMF).
* Тестирования корректности бесшовного роуминга на объекте при потере сигнала.
* Исследования оптических каналов управления (ИК-интерфейс): Регистрация, декодирование и воспроизведение сигналов инфракрасных пультов (бытовая техника, системы освещения) для анализа уязвимости устаревших протоколов к атакам повторного воспроизведения (Replay Attack).

3. Аппаратная платформа

Программный комплекс оптимизирован для работы в условиях ограниченных ресурсов встраиваемых систем. Вся логика низкоуровневой обработки сигналов и взаимодействия с трансивером реализована на базе однокристальной системы (SoC) — микроконтроллера ESP8266.
